Some use cases of Doom Emacs

  1. Using eshell to quickly switch between code and shell

    Installing the eshell package in the init.el of doom emacs give access to the keybinding (SPC o e) which toggles open the eshell pop-up window. This allows for quick access to a shell without having to switch to another dedicated terminal emulator.
