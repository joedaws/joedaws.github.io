Data Science

  1. Visualize the distribution of types of pokemon.

    • This could mean many things.
    • Do you want to compare types accross regions or generations?
    • Do you visually show how types are dispersed with in a region?
    • Do you want to make conclusions based on which types are the most numerous?

    This is a data visualization project where one can try to tell an insteresting story from geo-spatial data of pokemon and their types. The reason I think this project is an interesting portfolio project is that, the work to create the dataset which will eventually be visualized will have to be constructed. I think this will help one learn not only about data collection, but also cleaning! Finally, one can explore the landscape of geo-spatial data visualization tools! There are probably much cool projects involving Pokemon, so if you think of one, do that instead.
