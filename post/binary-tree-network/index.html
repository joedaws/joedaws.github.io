Binary Trees

Binary trees are both interesting from a purely graph theorectic point of view as well as in application. Their usefulness is evident by their wide use as a model for organization in a wide variety of applications. For example, they are used to describe directory hierachies in file systems, to store data so that it may be effeciently queried and to describe and organize the possible game states such as all possible chess matches.
